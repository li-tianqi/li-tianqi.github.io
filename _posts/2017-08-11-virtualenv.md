## 安装virtualenv
```
$ pip install virtualenv
```

## 创建环境
```
$ virtualenv venv_name
```

## 激活环境
```
$ source venv_name/bin/activate
```

## 退出环境
```
$ deactivate
```

## 删除环境
> 删除对应文件夹
