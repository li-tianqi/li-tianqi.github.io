平时习惯用python3，所以下面的安装过程都是基于python3
注：建议在虚拟环境中安装，virtualenv的安装及用法这里不介绍

1. 首先安装pip3 

$ sudo apt install python3-pip
2. 通过pip3在线安装django 
$ pip3 install django
3. 安装完成，通过python3测试一下 
$ python3
>>> import django
>>> django.get_version()
显示Django版本表示正确安装
